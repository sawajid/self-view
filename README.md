# Personal Website - Syed Abdul Wajid

A modern, professional personal website and digital resume built to showcase 16+ years of expertise in Power Sector IT Systems, Java Architecture, and Engineering Management.

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Vanilla CSS (Minimalist & Corporate Light Mode)
- **Icons**: React Icons (FaLinkedin, FaEnvelope, FaPhone, etc.)
- **Deployment**: GitHub Pages

## Features

- **Professional Summary**: Highlights core expertise in Java ecosystem and Power Systems domain.
- **Interactive Skills Grid**: Categorized view of programming languages, frameworks, databases, and domain standards.
- **Experience Timeline**: Detailed career progression from PwC India to Kalkitech, showcasing leadership and technical growth.
- **Certifications & Achievements**: Includes latest certifications in Generative AI, Electricity Derivatives, and Scrum.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile viewing.
- **Resume Download**: Integrated link to the latest PDF resume.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/self-view.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Deployment

This website is automatically deployed to GitHub Pages on every push to the `main` branch via a GitHub Actions workflow.

### Manual Deployment (Optional)

You can also deploy manually from your local machine:
```bash
npm run deploy
```

## Maintenance

To update the website content, modify the data in `src/constants.ts`. The UI will automatically reflect the changes.
